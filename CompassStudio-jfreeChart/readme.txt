
Brief:
    
    This folder is about the wrapper source code for jfreechart and swtgraphics2d.
    CompassStudio provide the default software library in the lib folder, you can 
    import into the wrapper source code. If you do not want to use the software 
    library provided by us,you can download it yourself.

Required:

    jfreechart		1.5.3
    swtgraphics2d	1.1.0
    
Download:

    https://mvnrepository.com/artifact/org.jfree/jfreechart/1.5.3
    https://mvnrepository.com/artifact/org.jfree/swtgraphics2d/1.1.0
