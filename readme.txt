CompassStudio is developed on the Eclipse platform, which uses the EPL license. We have released the source code of CompassStudio CDT according to the EPL agreement, but this is only the source code of CompassStudio CDT, you cannot directly use the source code to build CompassStudio.

CompassStudio CDT™ C/OpenCL Development Tools:
The CompassStudio CDT™ Project provides a fully functional C and OpenCL Integrated Development Environment based on the Eclipse platform(Eclipse-CDT). Features include: support for project creation and managed build for various Compass toolchains, standard make build, source navigation, various source knowledge tools, such as type hierarchy, call graph, include browser, macro definition browser, code editor with syntax highlighting, folding and hyperlink navigation, source code refactoring and code generation, visual debugging tools, including memory, registers, and disassembly viewers.
See also https://projects.eclipse.org/projects/tools.cdt and https://eclipse.org/cdt

Useage:
The recommended way to obtain CompassStudio CDT is to download the full version of Eclipse-CDT from the Eclipse site and compile the CompassStudio CDT source code as part of Eclipse-CDT.
Eclipse CDT can be installed into an existing Eclipse installation using this p2 URL: https://download.eclipse.org/tools/cdt/releases/latest/ (see how)
Downloads links for older versions are available in Downloads.
