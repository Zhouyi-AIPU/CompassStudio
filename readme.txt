CompassStudio CDT™ C/OpenCL Development Tools:
The CompassStudio CDT™ Project provides a fully functional C and OpenCL Integrated Development Environment based on the Eclipse platform (Eclipse CDT). 
The features include: 
• Support for project creation and manage build for various Compass toolchains and standard make build
• Source navigation, various source knowledge tools, such as type hierarchy, call graph, include browser, and macro definition browser 
• Code editor with syntax highlighting, folding and hyperlink navigation, source code refactoring and code generation
• Visual debugging tools, including memory, registers, and disassembly viewers
See also https://projects.eclipse.org/projects/tools.cdt and https://eclipse.org/cdt.

Usage:
The recommended way to obtain CompassStudio CDT is to download the full version of Eclipse CDT from the Eclipse site and compile the CompassStudio CDT source code as part of Eclipse CDT.
Eclipse CDT can be installed into an existing Eclipse installation using this p2 URL: https://download.eclipse.org/tools/cdt/releases/latest/ (see how)
Download links for older versions are available in Downloads.
